ПРОБЛЕМА 1: Бот крашится при старте
─────────────────────────────────────
Ошибка: "Cannot close a running event loop"
Решение: Pterodactyl не поддерживает, нужен VPS

ПРОБЛЕМА 2: Не входит в Steam
──────────────────────────────
Ошибка: "❌ Ошибка входа"
Причины:
- Неверный логин/пароль
- Нужен Steam Guard код
- IP заблокирован

Решение:
- Проверьте данные аккаунта
- Отключите Steam Guard
- Добавьте IP в whitelist

ПРОБЛЕМА 3: Dota 2 не запускается
──────────────────────────────────
Ошибка: "❌ Таймаут запуска Dota 2"
Причина: Недостаточно ресурсов / ограничения Docker

Решение: Нужен VPS с большими ресурсами

ПРОБЛЕМА 4: Лобби не создается
───────────────────────────────
Ошибка: "❌ Таймаут создания лобби"
Причина: API не отвечает / сетевые ограничения

Решение: Повторите или используйте VPS
